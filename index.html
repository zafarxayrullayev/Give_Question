Logical Operators

Logical operator nima?

Bu bizga bir nechta holatlarini birlashtirgan holatda bitta “true” yoki “false” qiymat hosil qilish uchun yordam beradi. Masalan: if(num > 0 && num % 2 == 0) { Console.Write($“{num} soni musbat va u juftdir”); } Bu yerda “num” sonini musbat va juft holatlarini birlashtirib tekshirayapmiz.

Logical operator turlari qanday?

1. && --> “and” 2. || --> “or” 3. ! --> “not”

( true && true || false ) natija qanday chiqadi?

True

( true || false && true || true ) natija qanday chiqadi?

True
