Logical Operators

Logical operator nima?

Bu bizga bir nechta holatlarini birlashtirgan holatda bitta “true” yoki “false” qiymat hosil qilish uchun yordam beradi. Masalan:
if(num > 0 && num % 2 == 0) {
Console.Write($“{num} soni musbat va u juftdir”);
}
Bu yerda “num” sonini musbat va juft holatlarini birlashtirib tekshirayapmiz.

Logical operator turlari qanday?

1. && --> “and” 2. || --> “or” 3. ! --> “not”

( true && true || false ) natija qanday chiqadi?

True

( true || false && true || true ) natija qanday chiqadi?

True

Switch operatori (tanlash operatori)

Switch nima uchun kerak?

Bizda bir qancha holatlar bo’lsin. Shu holatlar ro’y berganda “bunday” harakat bajarilsin degan narsani “shart operatori”da ham yozishimiz mumkin, lekin bu ko’proq kod, ko’proq vaqt oladi. Shuning uchun “switch” operatorini ishlatishimiz kerak bo’ladi.

break nima uchun kerak?

case bajarilib bo’lganda “switch” blokidan chiqib ketish uchun.

Switch Expressions

Switch Expressions haqida ma'lumot
